# Overview
Homie is a software to automate you home using low-budget hardware, like a RaspberryPi

# Requirements
 - PHP5.6 / PHP7
 - nodejs 0.10 - 4.1
 - npm
 - redis-server

# Installation
  - composer install
  - npm install
  - grunt exec:install
  - ./console user:create username password
  - ./console server:run
