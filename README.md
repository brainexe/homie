# Overview
Homie is a software to automate you home using low-budget hardware, like a RaspberryPi

# Installaltion
Checkout [this installation executable](https://github.com/brainexe/homie_install)

Manual way:
  - npm install
  - grunt exec:install

## Requirements
- PHP5.6 / PHP7
- nodejs 0.10 - 4.1
- redis server
